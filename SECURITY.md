SECURITY.md
Política de seguridad

Este repositorio forma parte de una investigación técnica aplicada a seguridad web y bug bounty ético.

El contenido está orientado a:

Análisis del comportamiento real de Unicode en parsers HTML y motores JavaScript.

Diferenciación entre ejecución sintácticamente válida y ofuscación visual.

Estudio técnico basado en especificaciones oficiales (Unicode / ECMAScript).

No se incluyen exploits contra objetivos específicos ni vectores dirigidos a sistemas reales. 


Alcance

Este repositorio no es:

Una herramienta de explotación automática.

Un framework ofensivo.

Un generador de payloads dirigido a sistemas reales.

Es una investigación técnica sobre comportamiento de parsers y ejecución semántica bajo Unicode.
